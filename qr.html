QR Code Generator

December 2023 Web Application

About the Project

A responsive web application that generates QR codes from user input. Users can create QR codes for URLs, text, contact information, and more. The application provides customization options for QR code colors and size.

Technologies Used

HTML5 CSS3 JavaScript QR Code API Netlify

Key Features

  • Generates QR codes from text, URLs, or other input
  • Customizable QR code colors and size
  • Download functionality for generated QR codes
  • Responsive design works on all devices
  • Clean, user-friendly interface
